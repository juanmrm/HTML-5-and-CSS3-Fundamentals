1 - Page Basics (ver first-page.html)
  
  Etiqueta raiz -> root tag.
  Etiqueta padre -> parent tag
  Etiquetas hijas -> nested tags

  Se pueden poner en mayusculas o minusculas.
  
  Autoclosing Tags (aquellos que no necesitan etiqueta de cierre)
  
  Los espacios, saltos de linea se ignoran y aparecen como un unico espacio

2 - span tag:
    Used to group inline-elements in a document. 
    Provides no visual change by itself.
    Provides a way to add a hook to a part of a text or a part of a document.

3 - div tag:
    Defines a division or a section in an HTML document.
    Used to group block-elements to format them with CSS.
